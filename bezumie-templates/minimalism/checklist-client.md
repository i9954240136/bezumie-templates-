# Чек-лист контента для клиента

## 📋 Что нужно предоставить ДО начала работы

### 1. Основная информация
- [ ] **Название компании** (как будет в логотипе)
- [ ] **Сфера деятельности** (чем занимаетесь)
- [ ] **Год основания** (опционально)

### 2. Тексты
- [ ] **О нас** (100-300 символов)
  > Пример: «Мы — команда профессионалов с 10-летним опытом. 
  > Работаем по всей России. Гарантия качества.»

- [ ] **Услуги/товары** (3-6 позиций)
  > Формат: Название + Цена + Описание (2-3 предложения)
  > 
  > Пример:
  > • Разработка сайта — от 10 000 ₽ — Срок 2-3 дня
  > • Дизайн логотипа — от 5 000 ₽ — Срок 1 день

- [ ] **Преимущества** (4 пункта)
  > Пример: Быстро, Недорого, Качественно, Надёжно

### 3. Контакты
- [ ] **Телефон** (в формате +7-XXX-XXX-XX-XX)
- [ ] **Email**
- [ ] **Адрес** (город, улица, дом)
- [ ] **Режим работы** (Пн-Пт: 9:00 - 18:00)
- [ ] **Соцсети** (ссылки, если есть)

### 4. Медиа
- [ ] **Фото** (3-5 штук)
  > Можно: фото офиса, команды, работ, товаров
  > Формат: JPG/PNG, любое качество (сожмём)

- [ ] **Логотип** (если есть)
  > Если нет — будет текстовый

### 5. Домен (опционально)
- [ ] **Есть домен?** (да/нет)
- [ ] **Если да:** название домена + где куплен
- [ ] **Если нет:** поможем выбрать (~200-900 ₽/год)

---

##  Сроки

| Этап | Срок |
|------|------|
| Получение контента | До начала работы |
| Разработка | 2-3 дня после получения контента |
| Правки | 3 бесплатные, далее 1 000 ₽/правка |
| Финальная сдача | После финальной оплаты |

---

## 💡 Советы

1. **Тексты:** Пишите просто, как для клиента (не для юристов)
2. **Фото:** Лучше свои, чем стоковые (доверие выше)
3. **Цены:** Указывайте «от X ₽» (защита от вопросов)
4. **Контакты:** Проверяйте перед отправкой (ошибки стоят заявок)
